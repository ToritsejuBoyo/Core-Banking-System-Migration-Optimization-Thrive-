# Core-Banking-System-Migration-Optimization-Thrive-

## 1. Project Overview
This project documents my direct involvement in the integration, migration support, and operational optimization of core banking systems at Thrive Microfinance Bank.

Scope included:
- API integration with third-party providers (Qore, BankOne, Interswitch)
- Migration coordination and data readiness validation
- Production issue identification and resolution
- Regulatory compliance implementation (CBN directives)
- Disaster recovery planning and system resilience
- System testing and validation (UAT)

---

## 2. System Environment

- Core Banking Platform: BankOne / Qore
- Integration Layer: API-based services (REST)
- Third-party Providers: Interswitch (iBaaS), ErrandPay
- Automation Stack: Microsoft 365 (Power Automate, Outlook, SharePoint)
- Reporting: General Ledger (GL), Financial Reports
